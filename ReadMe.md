# Task Management System

Система управления поручениями для малого и среднего бизнеса. Руководители создают задачи, назначают исполнителей, сотрудники получают уведомления и отчитываются о выполнении — всё через Telegram-бота.

## 🚀 Основные возможности

- **Telegram-бот** — создание задач, изменение статусов, уведомления
- **Ролевая модель** — owner, manager, employee
- **Статусная модель** — new → assigned → in_progress → review → completed
- **Асинхронная обработка** — Kafka для очередей и событий
- **Распределённые транзакции** — Saga для создания задачи
- **Аналитика** — отчёты по задачам и загрузке сотрудников

## 🚦 Статусы задач

| Статус        | Описание                    |
| ------------- | --------------------------- |
| `new`         | Создана, ожидает назначения |
| `assigned`    | Назначена исполнителю       |
| `in_progress` | В работе                    |
| `review`      | На проверке                 |
| `completed`   | Выполнена                   |
| `rejected`    | Отклонена                   |
| `overdue`     | Просрочена                  |

## 🧑‍💻 Роли

| Роль         | Возможности                                     |
| ------------ | ----------------------------------------------- |
| **owner**    | Полный доступ, управление организацией          |
| **manager**  | Создание задач, назначение, просмотр всех задач |
| **employee** | Свои задачи, изменение статуса                  |

### Настройка Telegram бота

Создать бота у @BotFather

Получить токен

Добавить в .env: TELEGRAM_TOKEN=your_token

### Первичная настройка

В системе уже есть тестовый администратор (создаётся миграцией):
Email: admin@example.com
Password: admin123

### Привязка Telegram:

Найти бота

/start

Ввести admin@example.com

### 📊 Мониторинг

Health check: curl http://localhost:50051/health

Логи: Structured logging (JSON)

Метрики: Prometheus (опционально)

### Graceful Shutdown

Все сервисы корректно завершают работу:

Перестают принимать новые запросы

Дожидаются обработки текущих

Закрывают соединения с БД, Redis, Kafka
